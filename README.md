# Hospital Information Management System
This is a course project for Database Design, Fall 2020 by Zhu Xiaoxuan & Shen Yunfei.

### Database: hims

| ------------- | ----------- | ------------------------------------ |
| <u>**id**</u> | int(20)     | 1                                    |
